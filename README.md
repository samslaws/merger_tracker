# merger_tracker
Scripts to update the merger tracker

Step 1: Download the most recent CSV from the Merger Tracker Google Sheet <br>
Step 2: Run merger_combine.py with the CSV in the same folder <br>
Step 3: Run data_cleaner.py inputting the name of the file merger_combine.py generated <br>
Step 4: Update the js tracker file to point to the new csv file generated by data_cleaner.py <br>
Step 5: Upload the js and csv to newsroom <br>
Step 6: Update the source code and change the date updated, the number of mergers for that year, and the js file being pointed to.
